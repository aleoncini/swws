Serverless Workflow WorkShop - Home page

This is an HUB of mock services that can be used in a serverless workflow

URP Protocollo
api service

servizio che restituisce una stringa randomica (num. di protocollo) di N caratteri

URL [GET]: /urp/protocollo/{size}

PATH PARAM: size = il numero di caratteri randomici della stringa di ritorno
Verifica Codice Fiscale
api service

servizio che effettua una verifica (di primo livello, ovvero solo un controllo formale del formato) del codice fiscale immesso

URL [GET]: /cofis/isvalid/{cf}

PATH PARAM: cf = il codice fiscale da verificare
Log
api service

Semplice sistema di logging remoto

URL [POST]: /logs

Json object Log: {"message":"your message","level":"info|warning|error"}
Create Cloud Event
User Interface

Servizio web con una interfaccia che permette di creare un Cloud Event

URL: /cloudevent.html
Approve workflow requests
User Interface

lista di pratiche che in un workflow richiedono approvazione

URL: /approvals.html
All Workflows
User Interface

lista di tutte le pratiche create

URL: /workflows.html
